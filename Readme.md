# library-mgmt-api
A REST API written in Django Rest Framework.

## Technologies used

- [Django](https://www.djangoproject.com/)
- [Django Rest Framework](https://www.django-rest-framework.org)
- [Swagger](https://swagger.io/)



    ### THE ENDPOINTS ARE:
    
    For the API Documentation please visit:
    'doc/'   # Swagger UI
    'redoc/'  # Swagger UI
